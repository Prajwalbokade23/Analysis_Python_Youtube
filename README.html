YouTube Sentiment Analysis and Visualization Project

Overview

This project uses Python for performing sentiment analysis, emoji analysis, and various visualizations on YouTube comments.

Key Features

  • Sentiment Analysis: Classify user comments into positive, neutral, and negative sentiments.
  • Word Cloud Analysis: Generate word clouds for both positive and negative comments.
  • Emoji Analysis: Visualize the most frequently used emojis in the comments.
  • Visualization: Leverage Python libraries like Plotly, Matplotlib, and Seaborn for graphical representations.

Challenges

1. Loading CSV Data

Loading data from local files might cause errors like ParserError. Use the on_bad_lines='skip' parameter to handle malformed lines in CSVs.

comment = pd.read_csv(r'P:\Data_Analysis\Youtube/UScomments.csv', on_bad_lines='skip')

2. Sentiment Analysis

Use the textblob library for sentiment analysis. Text polarity is calculated to determine whether the sentiment is positive, negative, or neutral.

from textblob import TextBlob
polarity = [TextBlob(comment).sentiment.polarity for comment in comments["comment_text"]]
comments['polarity'] = polarity

3. Word Cloud Generation

Generate word clouds from comments to identify the most frequent words. Install the wordcloud package and visualize it using matplotlib.

from wordcloud import WordCloud
positive_wordcloud = WordCloud().generate(' '.join(comments_positive['comment_text']))
plt.imshow(positive_wordcloud)
plt.axis('off')
plt.show()

4. Emoji Analysis

Extract emojis from the text and visualize the most frequent emojis using the emoji library and plotly.

import emoji
all_emojis = [char for comment in comments['comment_text'] for char in comment if char in emoji.EMOJI_DATA]

Installation

  1. Clone the repository: 
    git clone https://github.com/your-username/Youtube_Sentiment_Analysis.git
  2. Navigate to the project directory: 
    cd Youtube_Sentiment_Analysis
  3. Install dependencies using requirements.txt (if available): 
    pip install -r requirements.txt

Running the Project

  1. Open the project in Jupyter Notebook: 
    jupyter notebook
  2. Run the analysis in the youtube_analysis.ipynb notebook to execute various tasks such as sentiment analysis, emoji analysis, and visualizations.
